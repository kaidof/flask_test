API endpoints
Return product list
Request params:
  • page - page that you need (int), default is 1
  • per_page - how many items per page (int), default is 10
curl -XGET localhost:5000/products curl -XGET localhost:5000/products?per_page=5&page=1
Return product by id curl -XGET localhost:5000/product/<PRODUCT_ID(UUID)>
Change product
Can change only name or price parameters. curl -XPATCH localhost:5000/product/<PRODUCT_ID(UUID)> -H "Content-Type: application/json; charset=UTF-8" -d '{"name":"NEW_NAME","price":4.95}'
Return order list
Request params:
  • page - page that you need (int), default is 1
  • per_page - how many items per page (int), default is 10
curl -XGET localhost:5000/orders curl -XGET localhost:5000/orders?per_page=5&page=1
Return order by id curl -XGET localhost:5000/order/<ORDER_ID(UUID)>
Return related products id list curl -XGET localhost:5000/related_products/<PRODUCT_ID(UUID)>
